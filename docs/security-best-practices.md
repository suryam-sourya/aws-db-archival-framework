# Security Best Practices

## Identity & Access Management

- Apply least privilege IAM policies
- Separate admin and automation roles
- Enable MFA for AWS accounts
- Rotate credentials regularly

## Storage Security

- Enable S3 bucket encryption
- Use AWS KMS managed keys
- Restrict bucket public access
- Configure lifecycle protections

## Network Security

- Deploy workloads inside private subnets
- Use Security Groups and NACLs
- Restrict database access by IP
- Enable VPC Flow Logs

## Monitoring & Auditing

- Enable AWS CloudTrail
- Configure CloudWatch alarms
- Monitor unauthorized access attempts
- Enable GuardDuty and Security Hub

## Backup & Recovery

- Validate archival backups regularly
- Test restore procedures
- Maintain disaster recovery workflows
