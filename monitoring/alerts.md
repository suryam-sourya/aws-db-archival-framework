# Monitoring Alerts

## Recommended CloudWatch Alerts

### Lambda Monitoring

- Lambda invocation failures
- Lambda timeout alerts
- Excessive Lambda retries

### Storage Monitoring

- S3 storage growth anomalies
- Failed Glacier lifecycle transitions
- Failed archive uploads

### Infrastructure Monitoring

- ECS task failures
- IAM policy changes
- EventBridge schedule failures

## Recommended Notification Channels

- Amazon SNS
- Email Notifications
- Slack Integrations
- PagerDuty

## Operational Recommendations

- Review alerts daily
- Configure escalation policies
- Maintain alert severity levels
- Track recurring failures
